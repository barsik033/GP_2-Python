# GP_2-Python

# api_estate.py

Скрипт предназначен для получения и сохранения данных об объектах недвижимости с API сервиса **inpars.ru**.  
Он автоматически запрашивает категории, регионы и объявления, применяет фильтры (Москва, продажа жилой недвижимости)  
и сохраняет результаты в Excel-файл.

### Структура данных в Excel

| Поле | Описание |
|------|-----------|
| `title` | Название объявления |
| `address` | Адрес объекта |
| `floor` | Этаж |
| `floors` | Количество этажей |
| `sq` | Общая площадь |
| `sqLand` | Площадь участка |
| `sqKitchen` | Площадь кухни |
| `cost` | Стоимость |
| `lat` | Географическая широта |
| `lng` | Географическая долгота |
| `name` | Имя контактного лица |
| `source` | Источник данных |
| `created` | Дата создания объявления |
| `region` | Регион |
| `city` | Город |
| `type` | Тип недвижимости |
| `section` | Раздел недвижимости |
| `category` | Категория объекта |
| `metro` | Ближайшая станция метро |
| `material` | Материал дома |
| `rooms` | Количество комнат |
| `sqLiving` | Жилая площадь |


# scraping_links.py

Скрипт предназначен для массового сбора ссылок на объявления о продаже квартир с сайта **cian.ru**  
одновременно по нескольким районам Москвы. Работает через Selenium, автоматически переходит по страницам (1–54)  
и сохраняет результаты для каждого района и общий список в виде CSV-файлов.


# nchange_scrap.py

Скрипт предназначен для автоматического сбора данных с ссылок, которые получились на предыдущем этапе.

### Структура данных в CSV

| Признак | Описание |
|----------|-----------|
| `title` | Название объявления, краткое описание объекта |
| `address` | Адрес квартиры или жилого дома |
| `floor` | Этаж, на котором находится объект |
| `floors` | Общее количество этажей в доме |
| `sq` | Общая площадь объекта, м² |
| `sqLand` | Площадь земельного участка, если есть |
| `sqKitchen` | Площадь кухни, м² |
| `cost` | Цена объекта в рублях |
| `lat` | Географическая широта |
| `lng` | Географическая долгота |
| `name` | Имя продавца или агентства |
| `source` | Источник публикации (сайт) |
| `created` | Дата создания или обновления объявления |
| `region` | Регион, где расположен объект |
| `city` | Город или населённый пункт |
| `type` | Тип недвижимости (квартира, дом и т.д.) |
| `section` | Раздел недвижимости (вторичка, новостройка и т.п.) |
| `category` | Категория объекта (жилая, коммерческая и т.д.) |
| `metro` | Ближайшая станция метро |
| `material` | Материал здания (панель, кирпич и т.д.) |
| `rooms` | Количество комнат |
| `sqLiving` | Жилая площадь, м² |
